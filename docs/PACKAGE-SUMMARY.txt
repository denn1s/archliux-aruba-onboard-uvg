ARUBA ONBOARD UVG - PACKAGE SUMMARY
====================================

What: Automated WiFi enrollment package for UVG CAMPUS CENTRAL (Arch Linux)
Why: UVG uses 802.1X EAP-TLS which requires certificates and complex configuration
Goal: One-command install, click enrollment link, automatic connection

FILES IN THIS PACKAGE
=====================
PKGBUILD                      - Arch package build script
Aruba_Onboard_Installer.deb   - Original Ubuntu package (repackaged)
aruba-onboard-wrapper         - Script that manages enrollment process
aruba-enrollment-helper       - Script that auto-configures NetworkManager
aruba-onboard.desktop         - Desktop integration (arubadp:// protocol)
wifi-interface.service        - Fix for Framework laptops (wlan0 creation)
aruba-onboard-uvg.install     - Post-install instructions

DOCUMENTATION
=============
README.md                     - Technical documentation & troubleshooting (in parent dir)
STUDENT-GUIDE.md              - Simple setup guide for students
PROFESSOR-GUIDE.md            - Distribution & support guide
TESTING-CHECKLIST.md          - Testing checklist before distribution

STUDENT WORKFLOW
================
1. Install: makepkg -si
2. Enroll: Click enrollment link from university portal
3. Connect: nmcli connection up "UVG CAMPUS CENTRAL"

AUTOMATION FEATURES
===================
✓ Extracts and repackages Ubuntu .deb for Arch
✓ Registers arubadp:// protocol handler
✓ Manages onboard-srv + onboard-ui lifecycle
✓ Captures PKCS#12 password from D-Bus automatically
✓ Auto-configures NetworkManager connection with all parameters
✓ Includes compatibility fixes for hardware (e.g. Framework laptops)
✓ Enforces NetworkManager compatibility
✓ Shows success/failure notifications

KEY INSIGHT
===========
The university network requires NetworkManager + wpa_supplicant.
This package enforces the working configuration.
